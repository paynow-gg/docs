# Customer

<Row>
  <Col>
    A Customer represents a user in your store.
  </Col>
  <Col sticky>
  
    ```bash {{ title: 'Endpoints' }}
      GET /store/customer
    ```
  </Col>
</Row>

---

## The Customer object

<Row>
  <Col>
    <Properties>
      <Property name="id" type="flake">
        id of the customer
      </Property>
      <Property name="store_id" type="flake">
        id of the store this customer belongs to
      </Property>
      <Property name="steam_id" type="?string">
        the SteamID64 of this customer
      </Property>
      <Property name="steam" type="?steam profile object">
        the basic Steam profile information for this customer
      </Property>
      <Property name="name" type="?string">
        the name of the customer
      </Property>
      <Property name="created_at" type="timestamp">
        timestamp of when the product was created.
      </Property>
      <Property name="updated_at" type="?timestamp">
        timestamp of when the product was last updated.
      </Property>
    </Properties>

    ### Steam Profile Structure

    <Properties>
      <Property name="id" type="string">
        the profile's SteamID64
      </Property>
      <Property name="name" type="?string">
        the profile's display name
      </Property>
      <Property name="avatar_url" type="?string">
        the profile's avatar url
      </Property>
    </Properties>
  </Col>
