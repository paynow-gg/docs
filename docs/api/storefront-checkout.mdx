# Storefront Checkout

<Row>
    <Col>
        A checkout represents a customer's virtual shopping cart at the point of purchase.
    </Col>
    <Col sticky>
        <Endpoints>
            <Endpoint method="POST" href="#create-checkout" url="/checkouts" />
        </Endpoints>
    </Col>

---
## Create Checkout {{ tag: 'POST' }}

This endpoint creates a new checkout.

<Row>
    <Col>
        <Fields>
            <Field name="customer_id" type="flake">
                id of the customer this checkout belongs to
            </Field>
            <Field name="subscription" type="boolean">
                boolean indicating if this checkout is for a subscription
            </Field>
            <Field name="lines" type="[]CreateCheckoutRequestLine" typeHref="#checkout-line-object-structure">
                list of items in checkout
            </Field>
        </Fields>

        ## Checkout line object structure

        <Fields>
            <Field name="product_id" type="flake">
                id of the product
            </Field>
            <Field name="gift_to" type="nullable (optional)" typeHref="#checkout-gift">
                gift using a platform, currently only steam is supported
            </Field>
            <Field name="gift_to_customer_id" type="flake">
                id of the customer to gift to
            </Field>
            <Field name="quantity" type="integer">
                the quantity (count) of products in this line
            </Field>
        </Fields>

        ## Checkout Gift
        <Fields>
            <Field name="platform" type="platform">
                platform used to gift, steam is currently the only supported platform
            </Field>
            <Field name="id" type="integer">
                steam id of the customer to gift to
            </Field>
        </Fields>
    </Col>

    <Col sticky>
