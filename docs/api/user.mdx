# User

<Row>
  <Col>
    Users are a core entity in PayNow. They can create stores, receive payouts, and everything in between.

    Users have their own [type of authentication](api/reference#types-of-authentication).
  </Col>
</Row>

---

## The User object

<Row>
  <Col>
    <Fields>
      <Field name="id" type="flake">
        id of the user
      </Field>
      <Field name="email" type="string" nullable>
        email of the user.
        Cannot be seen by other users.
      </Field>
      <Field name="first_name" type="string">
        first name of the user
      </Field>
      <Field name="last_name" type="string">
        last name of the user
      </Field>
      <Field name="created_at" type="timestamp">
        timestamp of when the product was created
      </Field>
    </Fields>
  </Col>
  <Col sticky>
    ```json {{ title: 'The User object' }}
    {
        "id": "11084680073842688",
        "email": "example@paynow.gg",
        "first_name": "Example",
        "last_name": "Exampleson",
        "created_at": "2023-04-13T01:26:04Z"
    }
    ```
  </Col>
</Row>
