# API Key

<Row>
  <Col>
    An API Key lets you programatically manage your [store](api/store) without going through the dashboard.

    API keys can be created in the [API Keys section on the dashboard](https://paynow.gg/dashboard/api-keys).
    
    Only the [user authentication](reference#types-of-authentication) can call the API Key routes.
  </Col>
